// REASON FOR USING</br>


Diese Dateien helfen deine Server Performance zu verbessern.</br>


// Verwendung und Funktion</br>


Performance Binary = Diese EXE muss durch die arma3server.exe ersetzt werden. Danach ganz normal starten</br>


Server.cfg = Diese ServerCFG ist ein Beispiel mit den neusten Variablen und Parametern. Dadurch wird eine höhere Sicherheit gewährleistet</br>


Basic.cfg = Mit dieser Datei werden die Network Settings Definiert. Jeder Server benötigt eine für sich Optimierte Datei !</br>


bans.txt = Hier sind vordefinierte Bans für Hacker / ScriptKiddys eingetragen. Hierbei wird die Sicherheit des Servers erhöht.</br>


// INSTALLATION</br>

server- basic.cfg in den "config" Ordner deines Server ziehen. (In der Server.cfg müssen Daten wie "ServerName, Rcon-PW, MissionName gesetzt werden)</br>

bans.txt muss in den BattlEye Ordner eueres Server gezogen/ersetzt werden.</br>

Performance Binary wird mit der arma3server.exe erserzt. (Startparameter neu setzen)</br>
